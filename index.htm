Help
How to use the app
  1. Enter the lake name. The date is preloaded automatically with today’s date.
  2. Select the structure type, or choose Type your own.
  3. Tap Start logging.
  4. At each brush pile, stand still and tap Log location.
  5. The app keeps checking GPS until it has several fresh high-quality fixes, then saves the point automatically.
  6. Review the running list of saved points and their accuracies.
  7. Tap Done collecting to download a CSV and open the email workflow.
How to improve accuracy
  • Stand still while logging.
  • Give the phone a few seconds to settle before tapping log.
  • Keep the phone out in the open, not under metal covers or roofs.
  • Open sky usually gives the best results.
Enable location on iPhone
  1. Open Settings.
  2. Go to Privacy & Security → Location Services.
  3. Make sure Location Services is turned on.
  4. Find the browser being used, usually Safari or Chrome.
  5. Set location access to While Using the App.
  6. Turn on Precise Location.
Enable location in Safari
  1. Open the app webpage in Safari.
  2. When asked for location permission, tap Allow.
  3. If you already denied it, open Safari website settings for the page and allow location access, or clear the site settings and reload.
Enable location in Chrome on iPhone
  1. Make sure Chrome has location permission in iPhone settings.
  2. Open the app page and allow location when prompted.
Automatic email sending

A browser-only HTML app cannot securely send email from its own address by itself. To send automatically without the user’s email app, the app needs a backend service or email API.

Examples include a small server, serverless function, or an email provider API such as Microsoft 365, Gmail API, SendGrid, Postmark, or AWS SES. That backend would receive the CSV and send it directly to jehnsen.lebsock@nebraska.gov.
